# php-con-quiz

## Required
```
npm install -g firebase-tools
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```

### Functions
```
cd functions
```

### Functions Local Server
```
firebase serve
```
